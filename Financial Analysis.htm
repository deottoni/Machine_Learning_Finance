The Group

The Project

We've used a few Machine Learning models(LSTM, Random Forest, Arima) to try to understand which is the best approach to predict stocks.
The way we used to calculate which model is the best fit for predicting stocks is by calculating the Mean Squared Error (MSE).

Findings


The Best Model

According to the MSE numbers, Random Forest is the most accurate model to predict a stock closing. See the image to the right with the MSE scores for each model.

The Output

Our product takes a user input, and it returns the Adjusted Close value of each model comparing against the actual.
We also predict the next year of that stock. See an example below.
