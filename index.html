The Group

This is a project by group Fab5, and the members are Agustin, Godwin, Andry, Murtadha, and Andre.

The Project

We've used a few Machine Learning models(LSTM, Arima, Prophet) to try to understand which is the best approach to predict stocks.
The way we used to calculate which model is the best fit for predicting stocks is by calculating the Mean Squared Error (MSE).

Findings


Which Model is the Best?

We could not conclude on which model is the best, given that these models have different trends, and approaches. However all of them do provide a reasonable forecast of stock prices.

The Output

Our app takes the stock ticker as user input, and it returns the Adjusted Close value of each model.
